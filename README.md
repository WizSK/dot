# My dotfiles & Dwm install Things

## First run this commands to cp all the .files to ~/home
```bash
git clone https://github.com/sakibulxyz/files
cd files
cp .* ~/
# to copy .configs/files
cp -r .config/* ~/.config/
```
## To install dwm automaitcally use [jadu](https://github.com/sakibulxyz/jadu) script
```sh
git clone https://github.com/sakibulxyz/jadu
cd jadu
./jadu
```
